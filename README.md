## サービス概要
・Netflix、Huluのどちらかを選択、更にシチュエーション（テンション上げたい、チルしたい、ハラハラしたい）を選択し、その中からランダムでドラマを選んでくれるアプリ。
## ターゲット
・Netflix、Huluなどのサブスクを使用している人で、次に観るものが決まっていない、または、決めるのが面倒な人向けのサービスとなっております。
## なぜこのサービスを作ったのか？

今回このサービスを作ったのは、私自身、仕事終わりや休日にサブスクを利用してドラマやアニメをよく観るのですが、「次何観るか」の選択肢が多数ぎて、選ぶのが面倒だなと思うことが多々あります。また、私がドラマやアニメを選ぶ基準は基本的に、ポスター画像が「いい感じ」かどうかです。そのため、ランダムでドラマ、アニメが表示され、かつ、ポスター画像やタイトル、概要で「いい感じ」のものが出て来れば、数多くの選択肢から選ぶよりも、運任せで気楽にサブスクを利用出来ると思い作成しました。
